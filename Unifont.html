Better Unicode Viewer

Plenty of websites exist on the internet that will allow you to view all of unicode, but many of these are incomplete and/or don't include additional resources your browser needs to actually properly render all glyphs. This webpage forces the client to download the GNU Unifont and megamerge of notofonts, fonts that explicitly include all glyphs defined in Unicode. This ensures your browser is actually able to display all glyphs. You may need to disable browser extensions to avoid running out of memory loading this page.
